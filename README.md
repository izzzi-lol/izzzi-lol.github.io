# 📋 ТУТОРИАЛ: Написание досье для SCIPNET Terminal

> Полный справочник по всем тегам и возможностям системы.
> Версия: 30.05.2026

---

## 🗂️ Структура файлов

Каждое досье живёт в своей папке внутри `dossiers/`:

```
dossiers/
  └── ИМЯ_ПАПКИ/          ← это и есть ID для команды GET
       ├── dossier.txt     ← главный файл досье (обязательно)
       └── images/
            ├── фото.jpg
            └── схема.png
```

Команда для доступа: `get ИМЯ_ПАПКИ` или `get ключевое_слово`

## ⚙️ Системные теги (управление рендером)

Эти теги не выводятся — они управляют поведением рендерера.
Ставятся **на отдельной строке**.

### `!COM` — комментарий
Строка полностью игнорируется. Используй для заметок себе.

```
!COM Автор: izzzi | Версия 1.0 | Дата: 2026-05-28
!COM Не забыть добавить фото в папку images/
```

---

### `[SCROLLSPEED=мс]` — скорость анимации текста

Задержка между словами (или символами в CHARMODE) в миллисекундах.
Чем больше — тем медленнее. По умолчанию: `38`.

```
[SCROLLSPEED=15]   ← быстро, для экшн-сцен
[SCROLLSPEED=38]   ← стандарт
[SCROLLSPEED=100]  ← медленно, атмосферно
[SCROLLSPEED=200]  ← очень медленно, напряжённо
```

---

### `[CHARMODE]` / `[WORDMODE]` — режим анимации

```
[CHARMODE]
Этот текст будет печататься посимвольно, как печатная машинка...

[WORDMODE]
А этот — снова пословно (режим по умолчанию).
```

Хорошо комбинировать с `[SCROLLSPEED]`:

```
[CHARMODE]
[SCROLLSPEED=60]
СИГНАЛ ПРИНЯТ. РАСШИФРОВКА...
[WORDMODE]
[SCROLLSPEED=38]
```

---

### `[TIMER=с]` — пауза

Останавливает рендер на N секунд. Поддерживает дробные значения.

```
[TIMER=1]      ← пауза 1 секунда
[TIMER=0.5]    ← полсекунды
[TIMER=3]      ← драматическая пауза
```

---

### `[COLORCODES]` — цветовая тема досье

Переопределяет цвета элементов. Формат: `Ключ=#HEX; Ключ2=#HEX`.

```
[COLORCODES] Mainpage=#964B00; Quotes=#FFC0CB; Tables=#FFC0CB
```

| Ключ       | Что меняет                          |
|------------|-------------------------------------|
| `Mainpage` | Заголовки H1–H3, ссылки, маркеры   |
| `Quotes`   | Рамка и фон цитат `[QUOTE]`         |
| `Tables`   | Шапка таблиц `[TABLE6]`             |

Ставится **первой строкой** файла, до `[TITLE]`.

---

### `[DISABLE=TAGS]` / `[ENABLE=TAGS]` — отключить кастомные теги

```
[DISABLE=TAGS]
[COLOR=#ff0000]это не покрасится[/COLOR] — выведется как есть
[ENABLE=TAGS]
[COLOR=#ff0000]а это покрасится нормально[/COLOR]
```

### `[DISABLE=MD]` / `[ENABLE=MD]` — отключить Markdown

```
[DISABLE=MD]
**это не станет жирным**, _это не курсив_
[ENABLE=MD]
**а это уже жирный**
```

## 🏗️ Блочные теги (структура документа)

Занимают **отдельную строку**. Создают визуальные блоки.

### `[TITLE]` — название досье

Выводится крупным заголовком в стиле терминала.
Также используется как имя в результатах поиска `GET`.

```
[TITLE] ЮИ ГИНТО
```

---

### `[H1]` `[H2]` `[H3]` — заголовки разделов

Три уровня: H1 крупнее, H3 мельче.
Красятся в основной цвет темы (`Mainpage`).

```
[H1] Личные данные
[H2] Характер
[H3] Инцидент 2019-04-11
```

Также работает Markdown-синтаксис:

```
# Личные данные
## Характер
### Инцидент 2019-04-11
```

---

### `[CENTER]` — выравнивание по центру

Добавляется к заголовку или абзацу.

```
[H1] [CENTER] Личное дело
[CENTER] Этот абзац по центру.
```

---

### `[TABLE6]` / `[/TABLE6]` — таблица

Строки таблицы разделяются `||`. Первая строка — шапка.

```
[TABLE6]
Характеристика || Описание
Имя || Юи Гинто
Возраст || 34
Должность || Научный сотрудник
[/TABLE6]
```

> Инлайн-теги внутри ячеек работают (цвета, жирный, курсив и т.д.)

---

### `[IMAGE]` — изображение

```
[IMAGE] имя_файла.jpg || Подпись || позиция || масштаб
```

| Параметр     | Значения                     | По умолчанию |
|--------------|------------------------------|--------------|
| `имя файла`  | файл из папки `images/`      | обязательно  |
| `Подпись`    | текст под картинкой          | пусто        |
| `позиция`    | `right`, `left`, `full`      | `right`      |
| `масштаб`    | число: `0.8`, `1`, `1.5`...  | `1`          |

```
!COM Фото справа, стандартный размер:
[IMAGE] photo.jpg || Визуальная интерпретация || right

!COM Широкая иллюстрация по всей ширине:
[IMAGE] map.png || Карта зоны 02 || full || 1.2

!COM Маленькое фото слева:
[IMAGE] icon.png || Значок || left || 0.6
```

---

### `[QUOTE]` / `[/QUOTE]` — цитата

Оформляет текст как блок-цитату с цветной рамкой слева.

```
[QUOTE]
«Я просто подбирала рифму к слову "аномалия".»
— Юи Гинто, протокол допроса №4310
[/QUOTE]
```

> Внутри цитаты работают абзацы, инлайн-теги и даже списки.

---

### `[FOOTNOTE]` / `[/FOOTNOTE]` — сноска

Серый блок внизу с мелким текстом. Для оговорок и примечаний.

```
[FOOTNOTE]
- Персонаж не связан с Зоной 02.
- Документ проверен: Уровень доступа 3+.
[/FOOTNOTE]
```

---

### `[DANGER]` — предупреждение

Красный рамочный блок для опасных заметок.

```
[DANGER] ВНИМАНИЕ: УРОВЕНЬ УГРОЗЫ КЕТЕР. НЕСАНКЦИОНИРОВАННЫЙ ДОСТУП ЗАПРЕЩЁН.
```

---

### `.` — пустая строка (отступ)

Обычная пустая строка игнорируется рендерером. Чтобы добавить
визуальный отступ — используй точку на отдельной строке.

```
[H2] Раздел один
Текст раздела.
.
[H2] Раздел два
```

---

### `---` или `***` — горизонтальная линия

```
Текст выше линии.
---
Текст ниже линии.
```

---

## ✏️ Инлайн-теги (внутри строки)

Работают прямо в тексте абзаца, заголовка, ячейки таблицы и цитаты.

### Markdown

```
**жирный текст**
_курсив_
~~зачёркнутый~~
`моноширинный код`
```

### `[COLOR=#HEX]...[/COLOR]` — цвет текста

```
Статус: [COLOR=#00FF00]Активен[/COLOR]
Угроза: [COLOR=#FF0000]Кетер[/COLOR]
```

### `[BGCOLOR=#HEX]...[/BGCOLOR]` — фон текста

```
[BGCOLOR=#1a0a00]ЗАСЕКРЕЧЕНО[/BGCOLOR]
```
### `[SIZE=N]...[/SIZE]` — размер текста

Меняется размер текста.
```
[SIZE=12]Маленький текст[/SIZE]
[SIZE=64]ОГРОМНЫЙ ТЕКСТ![/SIZE]
```

### `[EFFECT=GLITCH;INTENSIVE=N]...[/EFFECT]` — глитч

Текст «ломается» с глитч-анимацией. `INTENSIVE` от `0.1` (слабо) до `2.0` (хаос).

```
[EFFECT=GLITCH;INTENSIVE=0.5]СИГНАЛ ИСКАЖЁН[/EFFECT]
[EFFECT=GLITCH;INTENSIVE=1.5]КРИТИЧЕСКАЯ ОШИБКА[/EFFECT]
```

### `[HREF=ссылка]текст[/HREF]` — ссылка

Открывается в новой вкладке.

```
Источник: [HREF=https://scp-wiki.net]SCP Foundation Wiki[/HREF]
```

### `[CMD="команда"][МЕТКА][/CMD]` — кликабельная команда

При клике анимированно вводит команду в терминал и выполняет её.

```
Связанное дело: [CMD="get 01005423"][→ ОБЪЕКТ-01005423][/CMD]
```

### `==текст==` — цензура (плашка)

Текст скрыт под чёрным прямоугольником. При наведении слегка проступает.

```
Личный идентификатор: ==YG-4310-DZETA==
Координаты объекта: ==[ДАННЫЕ УДАЛЕНЫ]==
```

---

## 📝 Списки

Используй `-` или `*` в начале строки. Поддерживают инлайн-теги.

```
- Первый пункт
- Второй пункт с [COLOR=#00FF00]зелёным текстом[/COLOR]
- **Жирный пункт**
```

## 🔀 Полный шаблон досье

```
!COM ============================================================
!COM  ДОСЬЕ: [ИМЯ ПЕРСОНАЖА]
!COM  Автор:          Версия:          Дата:
!COM ============================================================

[COLORCODES] Mainpage=#a200ff; Quotes=#a200ff; Tables=#a200ff
[TITLE] ИМЯ ПЕРСОНАЖА

[IMAGE] photo.jpg || Визуальная интерпретация || right

[H1] [CENTER] Личные данные
[TABLE6]
Характеристика || Описание
Имя || ...
Фамилия || ...
Возраст || ...
[/TABLE6]
.
[H1] [CENTER] Профиль
[TABLE6]
Характеристика || Описание
ID || ...
Должность || ...
Статус || [COLOR=#00FF00]Активен[/COLOR]
[/TABLE6]
.
[H1] [CENTER] Личное дело
[H2] [CENTER] Характер
Текст о характере...
.
[H2] [CENTER] История
[H3] [CENTER] Заголовок подраздела
Текст...
.
[QUOTE]
«Цитата персонажа.»
— Источник
[/QUOTE]
.
[DANGER] ПРЕДУПРЕЖДЕНИЕ: УРОВЕНЬ ДОСТУПА 4+
.
[H2] [CENTER] Связанные дела
[CMD="get другой-id"][→ ПЕРЕЙТИ К СВЯЗАННОМУ ДОСЬЕ][/CMD]
.
[FOOTNOTE]
- Примечание 1.
- Примечание 2.
[/FOOTNOTE]
```

---

## 💡 Советы

- **Первая строка** — всегда `[COLORCODES]`, если нужна тема. Иначе будет фиолетовый по умолчанию.
- **`[TITLE]`** нужен для поиска — без него досье не найдётся по имени.
- **`!COM`** не виден игроку — пиши туда всё служебное.
- **Масштаб изображений** — меняй `|| 0.8` если картинка слишком большая.
- **`[TIMER]` + `[CHARMODE]`** дают лучший эффект для напряжённых сцен.
- **Таблицы** работают как якоря страницы — ставь их до длинных текстов.
